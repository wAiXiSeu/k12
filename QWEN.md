# K12 中小学生阅读指导

## 项目概述

本项目是基于教育部《中小学生阅读指导目录（2020年版）》300种书目的阅读指导资料库。包含原始书目目录和按难度阶梯重新规划的阅读计划，覆盖小学、初中、高中三个学段。

## 目录结构

```
k12/
├── QWEN.md                              # 项目说明（本文件）
├── README.md                            # 难度阶梯阅读计划（主文档）
├── LICENSE                              # 许可证
└── books/
    ├── 中小学生阅读指导目录2020.md        # 教育部原始书目（300种）
    └── 中小学生阅读计划_难度阶梯版.md     # 难度阶梯版阅读计划
```

## 关键文件说明

| 文件 | 内容 | 规模 |
|------|------|------|
| `books/中小学生阅读指导目录2020.md` | 教育部2020年版原始书目，按学段和学科分类（人文社科/文学/自然科学/艺术） | 442行，300种书目 |
| `books/中小学生阅读计划_难度阶梯版.md` | 在原始书目基础上，按四级难度阶梯重新规划阅读路径，含阅读建议和节奏安排 | 709行 |

## 难度分级体系

- ⭐ **启蒙级**：图文结合、篇幅短小、情节直观，培养阅读兴趣
- ⭐⭐ **基础级**：文字为主、叙事清晰、主题明确，建立阅读习惯
- ⭐⭐⭐ **提高级**：结构复杂、思想深刻、需要思考，提升阅读能力
- ⭐⭐⭐⭐ **挑战级**：篇幅宏大、内涵丰富、需背景知识，深度阅读

## 内容组织

- **小学段**（110种）：按年级分组（1-2年级、3-4年级、5-6年级），每阶段标注阅读目标、阅读节奏和推荐用时
- **初中段**（100种）：按年级分组（7年级、8年级、9年级）
- **高中段**（90种）：按年级分组（高一、高二、高三）

## 使用说明

本项目为纯 Markdown 文档，无构建或运行步骤。可直接阅读 `README.md` 获取难度阶梯版阅读计划，或查阅 `books/` 目录下的原始书目和详细计划。
