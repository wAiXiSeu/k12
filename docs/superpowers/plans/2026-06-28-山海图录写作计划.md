# 《山海图录：天地灵珠》写作实施计划

> **执行者须知：** 本计划用于指导《山海图录：天地灵珠》全书 75 章的写作。每章约 1000 字，按卷顺序推进。每卷为一个独立任务，完成后提交。

**目标：** 完成全书 75 章的写作，总计约 7.5 万字，覆盖全部 171 种山海经异兽。

**架构：** 全书分 8 卷，每卷 7-12 章。采用接力叙事，每卷一位核心主角，卷末传递使命给下一卷新主角。主线为收集散落的三十六颗天地灵珠。

**参考资料：**
- 设计文档：`docs/superpowers/specs/2026-06-28-山海经异兽冒险书设计.md`
- 异兽资料：`books/山海经奇珍异兽汇总.md`

## 全局约束

- 每章字数：约 1000 字（800-1200 字弹性范围）
- 目标读者：8-12 岁，语言生动但不幼稚，情节有起伏
- 异兽特征：必须基于《山海经》原文记载，不可随意编造外形和能力
- 叙事风格：冒险为主，穿插幽默和温情，每章结尾留悬念或情感钩子
- 文件命名：`novel/第X卷_卷名/章节序号_章节标题.md`

## 执行建议

### 推荐执行方式

由于全书 75 章、约 7.5 万字，建议按卷分批执行：

1. **第一批（Task 0-2）**：目录结构 + 第一卷 + 第二卷（16章）
2. **第二批（Task 3-4）**：第三卷 + 第四卷（22章）
3. **第三批（Task 5-6）**：第五卷 + 第六卷（20章）
4. **第四批（Task 7-8）**：第七卷 + 第八卷（17章）
5. **收尾（Task 9）**：全书检查与 README

### 每卷写作流程

1. 重读设计文档中该卷的章节大纲
2. 查阅 `books/山海经奇珍异兽汇总.md` 中该卷涉及的异兽
3. 按步骤逐章写作
4. 每章写完后检查字数和质量
5. 全卷完成后通读检查
6. 提交 git

### 写作风格参考

- 语言：生动活泼，但不低幼。可以用成语和修辞，但要让 8-12 岁读者能理解
- 节奏：每章至少一个小高潮或悬念点
- 对话：异兽之间的对话要有个性区分
- 描写：异兽外形描写要基于原文，但可以用儿童能理解的方式表达
- 主题：不说教，通过情节自然传达价值观
