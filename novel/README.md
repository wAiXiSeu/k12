# 《山海图录：天地灵珠》

> 一本面向 8-12 岁读者的山海经异兽冒险小说，以西游记风格的接力叙事串联全部 171 种异兽。

---

## 故事简介

远古之初，盘古开天辟地后，灵气凝聚为**三十六颗天地灵珠**，镇守山海世界八方。灵珠之力维系着万物秩序——日月运行、四季更替、异兽灵性。

一日，被封印的四大凶兽挣脱束缚，击碎灵珠，天地秩序崩坏。上古瑞兽**白泽**将最后的灵力注入一卷空白的《山海图录》，将使命托付给勇敢的异兽们。

从南方的丹穴山到天地尽头的大荒，八位主角接力踏上寻找灵珠碎片的旅途。每一卷，一位新的主角接过使命，带领读者探索山海世界的奇妙与危险。

## 核心主题

- **每个生命都有独特的力量**：八位主角各有缺陷和局限，但正是这些"不完美"成为解决问题的关键
- **接力与信任**：使命的传递不是简单的交接，而是学会信任他人、放下控制
- **贪婪与知足**：饕餮代表的贪欲是永恒的威胁，知足是破解之道
- **混沌与秩序**：帝江代表的混沌不是邪恶，而是一种未被理解的状态
- **永不放弃**：精卫的填海精神贯穿全书

## 阅读指南

1. **按顺序阅读**：全书采用接力叙事，每卷结尾衔接下一卷
2. **关注异兽特征**：每种异兽的外形和能力都基于《山海经》原文
3. **体会主题**：每卷都有独立的主题（勇气、智慧、坚持、理解、知足、耐心、释放、担当）
4. **发现彩蛋**：前卷的角色会在后卷以"精神力量"的形式回归
